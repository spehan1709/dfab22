Assignment - Chess piece

To make the chess piece, I made a simple shape in 2d then used the revolve tool to make the base of the piece

For the main piece itself, I used curves and lines to make the outline of a horse piece.

I could have used the canvas tool to use an image found online to trace to make the shape, but I decided to draw it by hand.

Afterward, I used the extrude tool to complete the piece.

I also used the shell tool to make the model hollow to prevent the wastage of 3d printing material.
